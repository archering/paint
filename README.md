# paint
## Overview
***paint***

### 项目描述(description)
a command line tool for print folder structure in a REAPL


### 版本(version)
1.0.0

### 作者 (author)
ericever z00384109

### 如何使用(How to use?)

#### 安装(installation)
> 你需要先安装node，然后使用npm安装此工具

```
npm install -g http://z00384109@code-xa.huawei.com/z00384109/paint.git
``` 
 
#### getting start
 
(1)默认方式打印目录结构
```
paint  {folder_path}     
```
![paint default](/img/1.png)

(2)树状打印目录结构
```
paint -t folder 
```
![paint tree](/img/2.png)

(3) 打印目录层次结构
```
paint -p folder
```
![ske path](/img/3.png)
